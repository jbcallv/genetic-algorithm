# GeneticAlgorithm
Given a target phrase, this program finds uses a simple genetic algorithm
to efficiently find the given phrase.

# Biology
Concepts used were:
<ul>
<li>Single point chromosome crossover</li>
<li>Predefined fitness</li>
<li>Evolution</li>
</ul>
